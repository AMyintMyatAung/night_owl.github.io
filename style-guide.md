## Font

## Hero

Background Color: grayishBlue
Text Color: offWhite

```css
hero main {
    background: #1F2937;
    font-size: 48px;
    font-weight: extra-bold;
    color: #F9FAF8;
}

hero secondary, header link {
    font-size: 18px;
    color: #E5E7EB;
}

hero logo {
    font-size: 24px;
    color: #F9FAF8;
}
```
## Information

button,
